Booster Repetition Rate Limits
• Presently, the Booster is limited to an average rep. Rate of 2.5
Hz, which would only yield a small fraction of the desired
MiniBooNE intensity. The primary limitation is the pulsed
extraction septum magnet (MP02).
• A new power supply will go in in the next few weeks, which
will raise this rate to 4.5 Hz, which would give about half the
nominal MiniBooNE rate.
• A new magnet will go in during the January shutdown which
will allow the Booster to go to 7.5 Hz – enough for
MiniBooNE or NUMI.
• In order to run both MiniBooNE and NUMI, we need
– More cables for the extraction septum.
– Cooling for the RF cavities (or new RF cavities)
– A new “bump” magnet, which steers beam out during injection.

Configuring a New MacBook

Eric Prebys

These are some notes on configuring a new Mac, with MacOS 10.12 Sierra, assuming it came from the Fermilab pool with the standard setup.

Log into the cloud and all accounts.  Mail and calendars should be pulled in automatically.

Follow these instruction to enable the root account, then use "visudo" to add the line "prebys      ALL =  (ALL) ALL" under the similar line for root to enable "sudo".

Go here to install printers.

Install the following:
  • Firefox (then make default browser)
  • OneDrive (from App store, then sign in with services acccount)
  • Dropbox (need to log into online account, then go to "..." pulldown at the lower right to get client)
  • Google Drive
    • Before installing it:
      • > mkdir GoogleDrive
      • > ln -s GoogleDrive Google\ Drive
      • This workaround is needed for a ROOT bug
  • XCode Developer tools (app store). 
    • Start it.  It will download some additional components.
    • Then install appropriate "command line tools" from here with Apple credentials.
  • TextWrangler (from app store)
    • Also install command line tools
  • Kompozer
    • The page is a little confusing. Be sure to install the "development version" (0.8b3)!
  • Easyfind (from App store)
  • XQuartz
  • MacTex
  • TexWorks
    • After it's installed, go into "Preferences->typesetting" and add "/Library/TeX/texbin/" to the search path.  
  • ROOT
    • unpack file, then move the root tree to /usr/local,
    • create a file in /etc/paths.d called 'root' that contains the line '/usr/local/root/bin'
  • Install Office 2011 from disk and activate with code on package.
    • Must dowload 14.1.0 Service Pack by hand from here, because it will not automatically update.
    • Then do "Check for Updates" to automatically get the latest patches.
  • MathType
    • MathType corrupts the Symbol fond on Macs, so aftert it's installed, you need to search for "Font Book"  in Spotlight, and remove the Symbol->Symbol font.
  • Flip4Mac
  • VLC
  • Parallels
    • Recover VM files from backup or old computer
  • G4Beamline
    • Allow it to download and unpack GEANT4 datasets
  • Homebrew
    • Install homebrew version of python with "brew install python".  This makes sure all future paths are right.
    • Then do "hash -r python" or you will get errors.
  • pip
  • Do (at least) "brew install"
    • opencv (it will probably tell you to do "brew tap homebrew/science first")
    • gcc
    • scipy
    • matplotlib
  • Download ZOOM app, install, and log in
  • Use Self-service app to install Fermilab templates for Office 2011 and 2016

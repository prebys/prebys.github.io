Modifying ACNET Programs

Making this page so I can remember how to modify ACNET programs.

  • Connect using Cisco VPN client, even if at FNAL!
  • Use 'kinit' to get  kerberos ticket
  • Start a console using 'CNSRUN' app.
  • Must connect to the ACNET cluster in two steps
    • > ssh -XKA outback.fnal.gov
    • outback$ ssh -XKA clxNN.fnal.gov (NN = 1 to 80)
  • Make a local copy of the ACNET code
    • Go to page with application
    • Use "Cmnds->Display Identifiers" to get PA number for app
    • create a paNNNN subdirectory in the ~/mecca/ directory
    • > cd ~/mecca/paNNNN
    • > mecca --copy paNNNN
  • This will make a copy of all the files associated with the PA
  • Use gedit to edit files
  • Load a test copy of the program with
    • > mecca --dev
  • Load the final copy with
    • > mecca --commit
  • Close down console with "Shutdown Console" on "Utilities" window

Notes on using CNL Plotter to make big poster

The paper width is 36" and can only be accessed from the Windows server

  • User PowerPoint to create file
    • Set "Page Setup" to create a poster either 36" wide or high
  • Print,
    • use "Paper Size->Manage Custom Paper Sizes" to create the proper size
    • Doe "PDF->Open in Preview"
  • From Preview, click "Save..."
  • Log into "Crocker Computer"
  • Double-click on PDF file, which will open it in Adobe Acrobat PRO
  • Do "FIle->Properties" to verify that the size is correct
  • Do "File->Print"
    • Select "Computer Room Plotter"
    • Click "Properties->Custom Paper Size", enter sizes and click OK
    • Select "Fit" under "Page Size and Handling"
    • Verify that the size is right in the preview
    • Click "Print"
